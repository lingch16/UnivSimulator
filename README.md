# 大学模拟器/UnivSimulator v0.97

# 游戏方式：


1. 在线访问 https://lingch16.github.io/UnivSimulator/
2. 下载本仓库（除licence和readme以外的）所有文件，确保处于同一文件夹。点击index.html开始游戏。

可能有未知bug，欢迎报bug。     
也欢迎提出游戏平衡性相关的建议。
